# Aula 042 e 043 — Visão Geral do Capítulo

Nesta seção serão estudadas as **estruturas de repetição** (também chamadas de loops), fundamentais na lógica de programação para executar blocos de código **diversas vezes** enquanto uma condição for verdadeira.  

Essas estruturas permitem automatizar tarefas e reduzir código repetitivo, tornando os programas mais eficientes e legíveis.

---

## 43.1 Conteúdo que será estudado

- Estrutura `while` — repete enquanto a condição for verdadeira.
- Estrutura `for` — usada quando se sabe o número de repetições.
- Estrutura `do-while` — executa ao menos uma vez antes de testar a condição.
- **Testes de mesa e debug** passo a passo para entender a execução dos loops.
- **Exercícios práticos** para fixar o conteúdo.
